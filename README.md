# Conception et développement d’un système de gestion des étudiants en langage C avec persistance des données dans un fichier: Cas d’Ubuntu

# Objectifs:
1. Manipuler les fichiers en C 
2. Comprendre la gestion dynamique de la mémoire en C
3. Concevoir un mini-système CRUD (Create, Read, Update, Delete) en C

# Fonctionnalités attendues
1. Ajouter un étudiant
2. Afficher tous les étudiants
3. Rechercher un étudiant
4. Modifier un étudiant
5. Supprimer un étudiant
